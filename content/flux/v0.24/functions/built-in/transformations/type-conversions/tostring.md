The `toString()` function converts a value to a string.

_**Function type:** Type conversion_  
_**Output data type:** String_

```js
toString()
```

## Examples
```js
from(bucket: "telegraf")
  |> filter(fn:(r) =>
    r._measurement == "mem" and
    r._field == "used"
  )
  |> toString()
```

## Function definition
```js
toString = (tables=<-) =>
  tables
    |> map(fn:(r) => string(v: r._value))
```
